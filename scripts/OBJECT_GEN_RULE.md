# Object Generation Rule (ClawGotchi)

When creating a new object icon, use Gemini model `nano-banana` with this prompt template:

`Pixel art object icon, top-down 2D, transparent background PNG. Object: ____ Size: 64x64 pixels. Crisp pixels, consistent palette with the room background, no text, no shadow blur.`

Output requirement:
- Transparent PNG
- 64x64
- Pixel-crisp edges
